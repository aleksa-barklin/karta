Карта наблюдения

Выберите возрастную категорию ученика:
10–12 лет
13–15 лет
Процедура оценки: По каждому параметру карты (например, «Человек», «Эмоционально-регулятивная компетенция»):
1. Ознакомьтесь с описанием всех трёх уровней (Адаптационный, Активный, Просоциальный).
2. Выберите один уровень, наиболее точно соответствующий типичному поведению подростка.
3. Отметьте выбранный уровень и внесите его балл в графу «Балл».
Критерии выбора: Основой для оценки должны служить конкретные примеры поведения, которые вы наблюдали лично.
Принятие решения: Если в поведении присутствуют индикаторы из разных уровней, выберите уровень, которому соответствует большинство устойчивых проявлений.
